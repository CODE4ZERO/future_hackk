Problem

When a position is posted, a large number of candidates must go through the selection process. HR has to go through hundreds of applications in order to find the best candidate for the position.
This puts enormous strain on HR and makes it challenging to find the best candidate.
This process is costly to companies because it usually lasts a long time if many candidates apply.
For the candidate, it the simplest tasks become a pain to a candidate. Confusing platforms which are difficult to navigate, with slow performance make the process unexciting.

Solution Online Intergrated Platform

Develop an online platform that brings together employers and potential employees. The platform will improve candidate matching through the use of classifiers which are built using an algorithm that converts the personality traits to attributes that can be used to match a candidate to a position.
Using this platform will automate the process and improve the chances of finding a good candidate which is beneficial to both the employer and employee.
The process starts by the company creating a position and posting it on the platform. A candidate can set notifications for positions related to the one posted and will be notified when position is posted. The candidate can view the detailed information on the position, and apply if interested using their online profile. The system will match the candidate to the position and assign assessments to candidate. Employers can then view the information of the candidates and their performance ratings in a simple to use clean interface. They can proceed to select the candidates they interested in.

Benefits How it solves the problem


Improved candidate matching

Through the use of Support Vector Machines for the classificiation of candidates based on their behavioral and technical skills.

Streamlined Process

The interview process is done under one platform making it easier to manage the process.

Cost Saving

Activites are done under one hub with better management and less overhead.
